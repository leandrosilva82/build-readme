## Install

```bash
npm i -g build-readme
```

## Usage

Run the following command and follow the prompts to generate a README file.

```bash
build-readme
```

## Development & Testing

Install project dependencies locally and run linting:

```bash
npm i

npx eslint yourfile.js
```
